# Juego-en-python
Es un juego de texto en Python donde guías a Rapunzel por un castillo lleno de habitaciones, guardias y objetos. El jugador distribuye puntos de habilidades al inicio y luego explora un mapa de habitaciones conectadas. Tras escapar del castillo, enfrenta una segunda fase: explorar una matriz 6×6 del bosque para encontrar a Flynn.
